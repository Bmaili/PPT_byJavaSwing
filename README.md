## 使用：

​     运行环境推荐jdk11及以上，经测试jdk8布局有异。

-   方式一：运行bat文件（Windows下）,或命令行直接运行项目jar包。（推荐）

-   方式二：对项目源文件进行编译后再运行。项目启动类：ProjectStart.java
